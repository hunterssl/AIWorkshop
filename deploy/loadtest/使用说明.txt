============================================================
  灵影平台 · 批量压力测试包（发给测试同事用）
============================================================

【这个包是干什么的？】
  模拟多台电脑同时向灵影平台发送「文生图」任务，
  统计从提交到生成完成要多少秒，结果保存在 results 文件夹。

【需要什么环境？】
  1. Windows 10/11
  2. 已安装 Python 3.8 或更高版本（安装时勾选 Add Python to PATH）
  3. 能访问到灵影服务器（同一局域网或可访问的 IP）

【使用步骤 — 3 步即可】

  第 1 步：解压整个文件夹到任意位置（路径不要有中文空格问题）

  第 2 步：用记事本打开 config.txt，修改：
    BASE_URL  → 服务器地址，例如 http://192.168.1.10:8188
    USERNAME  → 你在主界面登录的用户名，例如 paj
    USER_ID   → 本机名字，建议每台电脑不同，如 PC-A、PC-B
    COUNT     → 本机要发多少条任务，默认 10
    CONCURRENCY → 同时跑几条，默认 1（想加压可改成 2、3）

  第 3 步：双击「快速测试1条.bat」先测连通；
           正常后再双击「运行压测.bat」跑完整测试。

【结果在哪里？】
  results 文件夹里的 CSV 文件，用 Excel 打开即可。
  列说明：
    user_id      — 哪台电脑跑的
    duration_sec — 这条任务耗时（秒）
    status       — completed 表示成功
    prompt_id    — 任务 ID

【多台电脑同时测】
  把同一个文件夹复制到多台电脑，每台改 config.txt 里的 USER_ID 不同即可，
  可同时双击「运行压测.bat」。结果各自保存在各自电脑的 results 里。

【常见问题】

  Q: 提示找不到 Python？
  A: 安装 https://www.python.org/downloads/ ，安装界面勾选 Add to PATH

  Q: 连接失败 / HTTP 错误？
  A: 检查 BASE_URL 是否正确、服务器是否已启动、防火墙是否放行 8188 端口

  Q: 找不到用户？
  A: USERNAME 必须和主界面登录的用户名一致（不是密码）

  Q: 没有匹配的应用？
  A: 双击「查看服务器应用列表.bat」，把 APP_ID 填进 config.txt

  Q: 任务一直 timeout？
  A: 加大 config.txt 里的 TIMEOUT，或降低 CONCURRENCY

【文件说明】
  config.txt              — 配置文件（必改）
  运行压测.bat            — 正式压测
  快速测试1条.bat         — 连通性测试
  查看服务器应用列表.bat  — 查看可用工作流
  studio_load_test.py     — 测试脚本（无需单独运行）
  results/                — 测试结果输出目录
